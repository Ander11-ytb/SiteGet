# SiteGet
SiteGet: Baixador de arquivos para Windows XP+
Inclui siteget.exe(XP-friendly, standalone) e siteget_help.exe com todos os parâmetros.

(para executar o siteget.exe e siteget_help.exe, abra o CMD e vá para o caminho onde está o siteget.exe e siteget_help.exe e digite: 'siteget_help' para ajuda e siteget você usa quando aprender no siteget_help.exe)
